## 🚀 部署步骤

### 第一步：安装依赖

打开命令行，运行：

```bash
pip install pymysql requests
```

### 第二步：配置数据库和 GitHub

1. 复制配置文件：
   ```
   复制 config-db.json.example 为 config-db.json
   ```

2. 编辑 `config-db.json`，填入以下信息：

   ```json
   {
     "db": {
       "host": "192.168.0.12",
       "port": 3306,
       "user": "你的数据库用户名",
       "password": "你的数据库密码",
       "database": "ruo_yi"
     },
     "github": {
       "token": "你的GitHub Token",
       "repo": "xinglianyue/finance-tool"
     }
   }
   ```

### 第三步：获取 GitHub Token

1. 登录 GitHub
2. 点击右上角头像 → Settings
3. 左侧菜单 → Developer settings
4. Personal access tokens → Generate new token (classic)
5. 勾选 `repo` 权限（完整仓库访问）
6. 生成后复制 Token

### 第四步：运行脚本

```bash
python db-sync.py 20260601
```

**参数说明**：
- `20260601` = 2026年6月1日的数据
- `202606` = 2026年6月整月的数据

## ❓ 常见问题

### Q1: 提示 "HTTP 401"
**原因**：GitHub Token 无效或过期  
**解决**：重新生成 GitHub Token

### Q2: 提示 "连接数据库失败"
**原因**：数据库配置错误或网络不通  
**解决**：检查 config-db.json 中的数据库配置

### Q3: 提示 "日期没有找到任何数据"
**原因**：该日期在数据库中没有数据  
**解决**：确认日期格式正确，或该日期确实有数据

## ⏰ 自动化（可选）

如需每天自动同步，可以设置 Windows 任务计划程序：

1. 打开"任务计划程序" (Win+R → `taskschd.msc`)
2. 创建基本任务
3. 名称：`财务工具数据同步`
4. 触发器：每天 14:30
5. 操作：启动程序
   - 程序：`python`
   - 参数：`db-sync.py %date:~0,4%%date:~5,2%%date:~8,2%`
   - 起始于：脚本所在目录
