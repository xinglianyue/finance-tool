# 财务数据同步说明

## 📋 数据同步流程

```
MySQL数据库 (192.168.0.12)
    ↓ 技术人员运行脚本
db-sync.py 脚本
    ↓ 自动转换数据格式
GitHub (shared-data.json)
    ↓ 自动同步
网页 (index-new.html)
```

---

## 🚀 技术人员操作步骤

### 1. 首次配置（一次性）

```bash
# 安装Python依赖
pip install pymysql requests
```

### 2. 日常使用

```bash
# 同步指定月份数据
python db-sync.py 202604    # 同步2026年4月

# 同步指定日期数据
python db-sync.py 20260423  # 同步2026年4月23日
```

### 3. 自动同步（可选）

可以通过Windows任务计划程序实现每天自动同步：

1. 打开"任务计划程序" (Win+R → taskschd.msc)
2. 创建基本任务
3. 名称: 财务工具数据同步
4. 触发器: 每天 14:30
5. 操作: 启动程序
   - 程序: python
   - 参数: `db-sync.py %date:~0,4%%date:~5,2%%date:~8,2%`

## 📊 脚本功能

- ✅ 从MySQL数据库读取账单数据
- ✅ 支持3种商家类型（全量、城市、KA）
- ✅ 支持10个城市
- ✅ 自动转换数据格式
- ✅ 去重处理（同月份只保留最新数据）
- ✅ 推送到GitHub

## 🔧 故障排除

### 问题：提示"No module named 'pymysql'"

```bash
pip install pymysql requests
```

### 问题：提示"连接数据库失败"

检查：
1. 网络是否连通（能否访问192.168.0.12）
2. 数据库账号密码是否正确

### 问题：提示"推送失败"

检查：
1. GitHub Token是否有效
2. 网络是否正常
