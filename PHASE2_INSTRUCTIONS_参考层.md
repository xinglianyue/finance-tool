## 数据基础

每个Excel = 某一天的数据（1~当天累计值）

可用字段（0-based行号）：
- Row 2：城市名列
- Row 10：订单量
- Row 32：收入
- Row 36：代补金额
- Row 73：总成本
- Row 74：毛利

城市列：Col 5-14（共10城）

## 核心算法详解

### 从累计值算出每日实际值

```
Day[0] 实际值 = Day[0]累计值
Day[1] 实际值 = Day[1]累计 - Day[0]累计
Day[2] 实际值 = Day[2]累计 - Day[1]累计
...
```

### 周对比

取排序后最后14天：
- **本周** = Day[N-6] ~ Day[N] 实际值之和
- **上周** = Day[N-13] ~ Day[N-7] 实际值之和

### 根因四因素分解

```
① 订单量影响 = (本周日均订单 - 上周日均订单) × 上周单均UE
② 收入变化   = (本周日均收入 - 上周日均收入) × 本周订单量
③ 成本变化   = (本周日均成本 - 上周日均成本) × 本周订单量
④ 单均UE变化 = (本周单均UE - 上周单均UE) × 本周订单量
```

第一杀手：|①|、|②|、|③| 数值最大的那个。

### 综合排序

综合分 = 单均UE排名 × 60% + 毛利排名 × 40%

### 单日横评排序

综合分 = 单均UE排名 × 0.5 + 代补占比排名（倒序）× 0.3 + 单均成本排名（倒序）× 0.2

### 单日横评Tab配色

- 红：单均UE<0.1 或 代补占比>30%
- 橙：单均UE 0.1~0.3 或 代补占比20%~30%
- 绿：单均UE≥0.3 且 代补占比≤20%
